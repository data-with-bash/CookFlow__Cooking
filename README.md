# CookFlow

CookFlow is a single-file AI cooking planner micro app that turns a user's day context into a structured meal plan, grocery list, substitutions, and budget feasibility logic.

## Features

- Day-aware meal planning
- Pantry-based grocery gap detection
- Ingredient substitution suggestions
- Budget feasibility logic
- Mobile-friendly single-screen demo UI
- Built as a hackathon-friendly AI micro app

## Demo flow

1. Select your day type, budget, time, and cooking confidence.
2. Add pantry items and any dislikes or restrictions.
3. Click **Generate plan**.
4. Review meal plan, grocery list, substitutions, and budget logic.

## Files

- `index.html` — the full app in one file

## Run locally

Open `index.html` directly in your browser.

## Publish on GitHub Pages

1. Create a new public GitHub repository, for example `cookflow`.
2. Upload `index.html` and `README.md` to the root.
3. Go to **Settings > Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root` folder.
6. Save, then wait for GitHub Pages to publish the site.
